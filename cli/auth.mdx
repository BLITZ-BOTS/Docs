---
title: CLI Authentication
description: 'Login to Blitz on your device to run acount restricted commands'
---


### CLI Command
```bash Terminal
blitz auth
```
This will open a login prompt in your browser. Once logged in, you will not need to log back in.



### Saved Credentials
Your credentials are saved locally on your device
<CodeGroup>
```bash Windows
C:\Users\YourUsername\AppData\Roaming\.blitz
```
```bash Linux
/home/yourusername/.blitz
```
```bash macOS
/Users/yourusername/.blitz
```
</CodeGroup>
