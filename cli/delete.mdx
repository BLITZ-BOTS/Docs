## Prerequisites
<Warning>
You **must** be logged in to delete a plugin. [Learn how to login](/cli/auth).
</Warning>

- Own a plugin

## CLI Command
```bash Terminal
blitz delete <plugin_name>
```
<ParamField path="plugin_name" type="string" required>
  The name of your plugin
</ParamField>
