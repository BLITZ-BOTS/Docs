## Prerequisites
<Warning>
You **must** be logged in to publish a plugin. [Learn how to login](/cli/auth).
</Warning>

- ``blitz.config.yaml`` file with valid config.
- ``/event`` and/or ``/commands`` folder.

## CLI Command
<Warning>
For Linux based machines, please make sure you have **zip** installed before publishing.
</Warning>
```bash Terminal
blitz publish
```
