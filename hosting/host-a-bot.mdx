---
title: 'Host A Bot'
description: 'Learn how to host a bot.'
---

Blitz is partnered with [pella](https://pella.app) to provide you with free and straight forward hosting.

<Info>
  Your bot **must** be using the node.js runtime to be able to use pella as a host.
</Info>


<Steps>
  <Step title="Create Your Bot">
   Create your bot, install plugins and customise using the [Blitz Controller](/installation/controller) or the [Blitz CLI](/installation/cli)
  </Step>

  <Step title="Prepare Bots Files">
    Go to your bots saved files *(displayed under your bots name on the dashboard)* and zip it
  </Step>

  <Step title="Create A Server">
    Head over to [pella](https://pella.app) and create a new server
  </Step>

  <Step title="Uplaod Bots Code">
    Upload your bot's code when prompted to.
  </Step>
</Steps>
