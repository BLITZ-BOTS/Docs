# Installing A Plugin

## Blitz CLI

Using the blitz cli to install a plugin is as simple as running one command!

```bash
deno run -A jsr:@blitz-bots/cli install my_plugin
```

You can also pass a version along with the plugin name to install a specific version of a plugin.

```
deno run -A jsr:@blitz-bots/cli install my_plugin@1.2.3
```
